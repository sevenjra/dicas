# Para enfatizar uma palavras (<em>), usamos um * ou _:
Javascript é _cool_!
ou:

1
Javascript é *cool*!

Para dar forte ênfase em palavras (), você usa dois ** ou __:

1
**Da2k** é a pronúncia para **Daciuk**: DA-TWO-K!
ou

1
__Da2k__ é a pronúncia para __Daciuk__: DA-TWO-K!

- [ ] incomplete
- [x] complete
